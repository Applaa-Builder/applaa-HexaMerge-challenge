How to Play

  1. Swipe or use arrow buttons to move all tiles in one direction.
  2. When two tiles with the same number touch, they merge into one with their sum.
  3. Create a tile with the target value to win the level.
  4. If the grid fills up with no possible merges, the game is over.

View on Your Computer

You can view this app without a mobile device in two ways:
